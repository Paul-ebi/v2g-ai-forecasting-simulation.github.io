# Step 4 — Optimization Framework

This layer uses forecasts + system model outputs to schedule EV charging/discharging.

---

## Objectives
- Minimize operational cost  
- Maximize PV self-consumption  
- Protect EV battery health  
- Maintain grid stability  

---

## Optimization Problem

Approaches:

- Genetic Algorithm  
- Linear Programming  
- Reinforcement Learning  
- Model Predictive Control  

---

## Input Data
- Forecasted load  
- Forecasted PV  
- Forecasted EV availability  
- Simulink system parameters  

---

## Output
- Hourly EV charging plan  
- Battery dispatch  
- Net grid flow  
- Updated SOC profiles
